#### setOutline()


 void TabbedComponent::setOutline ( int newThickness ) 
 

Specifies the thickness of an outline that should be drawn around the content component.If this thickness is > 0, a line will be drawn around the three sides of the content component which don't touch the tabbar, and the content component will be inset by this amount.To set the colour of the line, use setColour (outlineColourId, ...).
