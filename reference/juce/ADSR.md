A very simple ADSR envelope class.To use it, call setSampleRate() with the current sample rate and give it some parameters with setParameters() then call getNextSample() to get the envelope value to be applied to each audio sample or applyEnvelopeToBuffer() to apply the envelope to a whole buffer.Do not change the parameters during playback. If you change the parameters before the release stage has completed then you must call reset() before the next call to noteOn().
