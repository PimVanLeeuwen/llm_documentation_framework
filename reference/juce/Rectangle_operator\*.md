#### operator\*()


template<typename ValueType > 
template<typename FloatType > 

 Rectangle Rectangle< ValueType >::operator\* ( FloatType scaleFactor ) const noexcept 
 

Returns a rectangle that has been scaled by the given amount, centred around the origin.Note that if the rectangle has int coordinates and it's scaled by a floatingpoint amount, then the result will be converted back to integer coordinates using getSmallestIntegerContainer().
