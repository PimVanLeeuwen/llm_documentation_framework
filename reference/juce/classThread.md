Encapsulates a thread.Subclasses derive from Thread and implement the run() method, in which they do their business. The thread can then be started with the startThread() or startRealtimeThread() methods and controlled with various other methods.This class also contains some threadrelated static methods, such as sleep(), yield(), getCurrentThreadId() etc.See alsoCriticalSection, WaitableEvent, Process, ThreadWithProgressWindow, MessageManagerLock
