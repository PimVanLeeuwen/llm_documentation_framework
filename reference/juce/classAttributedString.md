A text string with a set of colour/font settings that are associated with subranges of the text.An attributed string lets you create a string with varied fonts, colours, wordwrapping, layout, etc., and draw it using AttributedString::draw().Invariants:Every character in the string is a member of exactly one attribute.Attributes are sorted such that the rangeend of attribute 'i' is equal to the rangebegin of attribute 'i + 1'.See alsoTextLayout
