Represents a piano keyboard, keeping track of which keys are currently pressed.This object can parse a stream of midi events, using them to update its idea of which keys are pressed for each individual midi channel.When keys go up or down, it can broadcast these events to listener objects.It also allows key up/down events to be triggered with its noteOn() and noteOff() methods, and midi messages for these events will be merged into the midi stream that gets processed by processNextMidiBuffer().
