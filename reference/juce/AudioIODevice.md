Base class for an audio device with synchronised input and output channels.Subclasses of this are used to implement different protocols such as DirectSound, ASIO, CoreAudio, etc.To create one of these, you'll need to use the AudioIODeviceType class see the documentation for that class for more info.For an easier way of managing audio devices and their settings, have a look at the AudioDeviceManager class.See alsoAudioIODeviceType, AudioDeviceManager
