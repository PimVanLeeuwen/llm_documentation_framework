#### create5point0point4()


 static AudioChannelSet JUCE\_CALLTYPE AudioChannelSet::create5point0point4 ( ) static 
 

Creates a set for a 5.0.4 surround setup (left, right, centre, leftSurround, rightSurround, topFrontLeft, topFrontRight, topRearLeft, topRearRight).Is equivalent to: AAX\_eStemFormat\_5\_0\_4 (AAX).
