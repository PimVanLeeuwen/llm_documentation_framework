#### setDefaultSansSerifTypefaceName()


 void LookAndFeel::setDefaultSansSerifTypefaceName ( const String & newName ) 
 

Allows you to change the default sansserif font.If you need to supply your own Typeface object for any of the default fonts, rather than just supplying the name (e.g. if you want to use an embedded font), then you can instead call setDefaultSansSerifTypeface() with an object to use.
