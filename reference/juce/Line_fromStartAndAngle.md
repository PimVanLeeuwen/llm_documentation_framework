 static Line Line< ValueType >::fromStartAndAngle ( Point< ValueType > startPoint, ValueType length, ValueType angle ) staticnoexcept 
 

Creates a line from a start point, length and angle.This angle is the number of radians clockwise from the 12 o'clock direction, where the line's start point is considered to be at the centre.
