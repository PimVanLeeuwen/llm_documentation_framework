Enables draganddrop behaviour for a component and all its subcomponents.For a component to be able to make or receive draganddrop events, one of its parent components must derive from this class. It's probably best for the toplevel component to implement it.Then to start a drag operation, any subcomponent can just call the startDragging() method, and this object will take over, tracking the mouse and sending appropriate callbacks to any child components derived from DragAndDropTarget which the mouse moves over.Note: If all that you need to do is to respond to files being draganddropped from the operating system onto your component, you don't need any of these classes: you can do this simply by overriding FileDragAndDropTarget::filesDropped().See alsoDragAndDropTarget
