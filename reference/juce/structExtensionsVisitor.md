Create a derived implementation of this class and pass it to AudioPluginInstance::getExtensions() to retrieve formatspecific information about a plugin instance.Note that the references passed to the visit member functions are only guaranteed to live for the duration of the function call, so don't store pointers to these objects! If you need to store and reuse formatspecific information, it is recommended to copy the result of the function calls that you care about. For example, you should store the result of VST::getAEffectPtr() rather than storing a pointer to the VST instance.
