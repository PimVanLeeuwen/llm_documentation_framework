#### create7point1point2()


 static AudioChannelSet JUCE\_CALLTYPE AudioChannelSet::create7point1point2 ( ) static 
 

Creates a set for Dolby Atmos 7.1.2 surround setup (left, right, centre, leftSurroundSide, rightSurroundSide, leftSurroundRear, rightSurroundRear, LFE, topSideLeft, topSideRight).Is equivalent to: k71\_2 (VST), AAX\_eStemFormat\_7\_1\_2 (AAX), kAudioChannelLayoutTag\_Atmos\_7\_1\_2 (CoreAudio)
