A box with a small arrow that can be used as a temporary popup window to show extra controls when a button or other component is clicked.Using one of these is similar to having a popup menu attached to a button or other component but it looks fancier, and has an arrow that can indicate the object that it applies to.The class works best when shown modally, but obviously running modal loops is evil and must never be done, so the launchAsynchronously method is provided as a handy way of launching an instance of a CallOutBox and automatically managing its lifetime, e.g.void mouseUp (const MouseEvent&)
