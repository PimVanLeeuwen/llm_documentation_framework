#### getMonthName() [2/2]


 static String Time::getMonthName ( int monthNumber, bool threeLetterVersion ) static 
 

Returns the name of one of the months.Parameters

 monthNumber the month, 0 to 11 
 
 threeLetterVersion if true, it'll be a 3letter abbreviation, e.g. "Jan"; if false it'll return the long form, e.g. "January"
