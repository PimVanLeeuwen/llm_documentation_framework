A component that displays a strip of column headings for a table, and allows these to be resized, dragged around, etc.This is just the component that goes at the top of a table. You can use it directly for custom components, or to create a simple table, use the TableListBox class.To use one of these, create it and use addColumn() to add all the columns that you need. Each column must be given a unique ID number that's used to refer to it.See alsoTableListBox, TableHeaderComponent::Listener
