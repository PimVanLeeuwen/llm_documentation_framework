A utility class for fitting a set of objects whose sizes can vary between a minimum and maximum size, into a space.This is a trickier algorithm than it would first seem, so I've put it in this class to allow it to be shared by various bits of code.To use it, create one of these objects, call addItem() to add the list of items you need, then call resizeToFit(), which will change all their sizes. You can then retrieve the new sizes with getItemSize() and getNumItems().It's currently used by the TableHeaderComponent for stretching out the table headings to fill the table's width.
