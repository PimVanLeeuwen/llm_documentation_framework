#### addSectionHeading()


 void ComboBox::addSectionHeading ( const String & headingName ) 
 

Adds a heading to the dropdown list, so that you can group the items into different sections.The headings are indented slightly differently to set them apart from the items on the list, and obviously can't be selected. You might want to add separators between your sections too.See alsoaddItem, addSeparator
