This class is used when references to images need to be stored in ValueTrees.An instance of an ImageProvider provides a mechanism for converting an Image to/from a reference, which may be a file, URL, ID string, or whatever system is appropriate in your app.When you're loading components from a ValueTree that may need a way of loading images, you should call ComponentBuilder::setImageProvider() to supply a suitable provider before trying to load the component.See alsoComponentBuilder::setImageProvider()
