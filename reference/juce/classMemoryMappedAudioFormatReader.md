A specialised type of AudioFormatReader that uses a MemoryMappedFile to read directly from an audio file.This allows for incredibly fast randomaccess to sample data in the mapped region of the file, but not all audio formats support it see AudioFormat::createMemoryMappedReader().Note that before reading samples from a MemoryMappedAudioFormatReader, you must first call mapEntireFile() or mapSectionOfFile() to ensure that the region you want to read has been mapped.See alsoAudioFormat::createMemoryMappedReader, AudioFormatReader
