#### menuItemSelected()


 virtual void MenuBarModel::menuItemSelected ( int menuItemID, int topLevelMenuIndex ) pure virtual 
 

This is called when a menu item has been clicked on.Parameters

 menuItemID the item ID of the PopupMenu item that was selected 
 
 topLevelMenuIndex the index of the toplevel menu from which the item was chosen (just in case you've used duplicate ID numbers on more than one of the popup menus)
