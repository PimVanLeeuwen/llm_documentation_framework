#### createOnOffRamp()


 static std::function< float(float)> Easings::createOnOffRamp ( ) static 
 

Returns an easing function that reaches 1.0f when the input value is 0.5f, before returning to 0.0f when the input values reaches 1.0f.This is useful for making a repeating pulsation.
