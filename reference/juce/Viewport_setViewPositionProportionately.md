#### setViewPositionProportionately()


 void Viewport::setViewPositionProportionately ( double proportionX, 
 
 double proportionY ) 

Changes the view position as a proportion of the distance it can move.The values here are from 0.0 to 1.0 where (0, 0) would put the visible area in the topleft, and (1, 1) would put it as far down and to the right as it's possible to go whilst keeping the child component onscreen.
