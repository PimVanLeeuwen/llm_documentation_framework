A JSObject represents an owning reference to the underlying JS object, meaning it will remain valid even if a subsequent script execution deletes other handles to it.Objects of this class can be used to traverse the current object graph inside the specified Javascript engine.This is a lowlevel providing only operations that map directly to the underlying Javascript Object implementation. The JSCursor class generally provides a more convenient interface with functions that may fail based on the Javascript engine's current state.See alsoJSCursor
