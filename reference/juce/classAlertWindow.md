A window that displays a message and has buttons for the user to react to it.For simple dialog boxes with just a couple of buttons on them, there are some static methods for running these.For more complex dialogs, an AlertWindow can be created, then it can have some buttons and components added to it, and its enterModalState() method is used to show it. The value returned to the ModalComponentManager::Callback shows which button the user pressed to dismiss the box.See alsoThreadWithProgressWindow, Component::enterModalState
