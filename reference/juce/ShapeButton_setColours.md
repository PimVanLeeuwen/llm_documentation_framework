#### setColours()


 void ShapeButton::setColours ( Colour normalColour, 
 
 Colour overColour, 
 Colour downColour ) 

Set the colours to use for drawing the shape.Parameters

 normalColour the colour to fill the shape with when the mouse isn't over 
 
 overColour the colour to use when the mouse is over the shape 
 downColour the colour to use when the button is in the presseddown state
