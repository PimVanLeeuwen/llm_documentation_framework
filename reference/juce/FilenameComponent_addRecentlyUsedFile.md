#### addRecentlyUsedFile()


 void FilenameComponent::addRecentlyUsedFile ( const File & file ) 
 

Adds an entry to the recentlyused files dropdown list.If the file is already in the list, it will be moved to the top. A limit is also placed on the number of items that are kept in the list.See alsogetRecentlyUsedFilenames, setRecentlyUsedFilenames, setMaxNumberOfRecentFiles
