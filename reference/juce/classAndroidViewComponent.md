An Androidspecific class that can create and embed a View inside itself.To use it, create one of these, put it in place and make sure it's visible in a window, then use setView() to assign a View to it. The view will then be moved and resized to follow the movements of this component.Of course, since the view is a native object, it'll obliterate any JUCE components that may overlap this component, but that's life.
