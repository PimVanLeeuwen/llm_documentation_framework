Shows a filename as an editable text box, with a 'browse' button and a dropdown list for recently selected files.A handy component for dialogue boxes where you want the user to be able to select a file or directory.Attach an FilenameComponentListener using the addListener() method, and it will get called each time the user changes the filename, either by browsing for a file and clicking 'ok', or by typing a new filename into the box and pressing return.See alsoFileChooser, ComboBox
