#### didReorderRegionSequencesInMusicalContext()


 virtual void ARAMusicalContextListener::didReorderRegionSequencesInMusicalContext ( ARAMusicalContext \* musicalContext ) virtual 
 

Called after the region sequences are reordered in an ARA MusicalContext.Region sequences are sorted by their order index this callback signals a change in this ordering within the musical context.Parameters

 musicalContext The musical context with reordered region sequences.
