RSA public/private keypair encryption class.An object of this type makes up one half of a public/private RSA key pair. Use the createKeyPair() method to create a matching pair for encoding/decoding.If you need to use this class in conjunction with a compatible enc/decryption algorithm on a webserver, you can achieve the same thing in PHP like this:include ('Math/BigInteger.php'); // get this from: phpseclib.sourceforge.net
