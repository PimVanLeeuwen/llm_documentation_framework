A component that lets the user choose from a dropdown list of choices.The combobox has a list of text strings, each with an associated id number, that will be shown in the dropdown list when the user clicks on the component.The currently selected choice is displayed in the combobox, and this can either be readonly text, or editable.To find out when the user selects a different item or edits the text, you can assign a lambda to the onChange member, or register a ComboBox::Listener to receive callbacks.
