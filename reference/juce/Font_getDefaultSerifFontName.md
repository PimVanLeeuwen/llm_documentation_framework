#### getDefaultSerifFontName()


 static const String & Font::getDefaultSerifFontName ( ) static 
 

Returns a typeface font family that represents the default serif font.Note that this method just returns a generic placeholder string that means "the default
serif font" it's not the actual font family of this font.See alsosetTypefaceName, getDefaultSansSerifFontName, getDefaultMonospacedFontName
