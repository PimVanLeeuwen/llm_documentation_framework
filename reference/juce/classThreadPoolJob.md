A task that is executed by a ThreadPool object.A ThreadPool keeps a list of ThreadPoolJob objects which are executed by its threads.The runJob() method needs to be implemented to do the task, and if the code that does the work takes a significant time to run, it must keep checking the shouldExit() method to see if something is trying to interrupt the job. If shouldExit() returns true, the runJob() method must return immediately.See alsoThreadPool, Thread
