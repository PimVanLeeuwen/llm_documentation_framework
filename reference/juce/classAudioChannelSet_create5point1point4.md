#### create5point1point4()


 static AudioChannelSet JUCE\_CALLTYPE AudioChannelSet::create5point1point4 ( ) static 
 

Creates a set for a 5.1.4 surround setup (left, right, centre, LFE, leftSurround, rightSurround, topFrontLeft, topFrontRight, topRearLeft, topRearRight).Is equivalent to: kAudioChannelLayoutTag\_Atmos\_5\_1\_4 (CoreAudio).
