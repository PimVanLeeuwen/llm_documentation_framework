A toolbar component.A toolbar contains a horizontal or vertical strip of ToolbarItemComponents, and looks after their order and layout.Items (icon buttons or other custom components) are added to a toolbar using a ToolbarItemFactory each type of item is given a unique ID number, and a toolbar might contain more than one instance of a particular item type.Toolbars can be interactively customised, allowing the user to drag the items around, and to drag items onto or off the toolbar, using the ToolbarItemPalette component as a source of new items.See alsoToolbarItemFactory, ToolbarItemComponent, ToolbarItemPalette
