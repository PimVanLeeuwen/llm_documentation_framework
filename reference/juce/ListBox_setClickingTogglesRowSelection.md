#### setClickingTogglesRowSelection()


 void ListBox::setClickingTogglesRowSelection ( bool flipRowSelection ) noexcept 
 

If enabled, this makes the listbox flip the selection status of each row that the user clicks, without affecting other selected rows.(This only has an effect if multiple selection is also enabled). If not enabled, you can still get the same rowflipping behaviour by holding down CMD or CTRL when clicking.
