#### getDefaultMonospacedFontName()


 static const String & Font::getDefaultMonospacedFontName ( ) static 
 

Returns a typeface font family that represents the default monospaced font.Note that this method just returns a generic placeholder string that means "the default
monospaced font" it's not the actual font family of this font.See alsosetTypefaceName, getDefaultSansSerifFontName, getDefaultSerifFontName
