Used to receive callbacks on every vertical blank event of the display that the peer currently belongs to.On Linux this is currently limited to receiving callbacks from a timer approximately at display refresh rate.This is a lowlevel facility used by the peer implementations. If you wish to synchronise Component events with the display refresh, you should probably use the VBlankAttachment, which automatically takes care of listening to the vblank events of the right peer.See alsoVBlankAttachment
