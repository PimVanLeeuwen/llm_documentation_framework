#### create7point1point6()


 static AudioChannelSet JUCE\_CALLTYPE AudioChannelSet::create7point1point6 ( ) static 
 

Creates a set for Dolby Atmos 7.1.6 surround setup (left, right, centre, leftSurroundSide, rightSurroundSide, leftSurroundRear, rightSurroundRear, LFE, topFrontLeft, topFrontRight, topSideLeft, topSideRight, topRearLeft, topRearRight).Is equivalent to: k71\_6 (VST), n/a (AAX), n/a (CoreAudio)
