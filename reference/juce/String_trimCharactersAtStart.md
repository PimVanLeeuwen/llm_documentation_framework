#### trimCharactersAtStart()


 String String::trimCharactersAtStart ( StringRef charactersToTrim ) const 
 

Returns a copy of this string, having removed a specified set of characters from its start.Characters are removed from the start of the string until it finds one that is not in the specified set, and then it stops.Parameters

 charactersToTrim the set of characters to remove. 
 



See alsotrim, trimStart, trimCharactersAtEnd
