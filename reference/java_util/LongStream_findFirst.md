#### findFirst

```
OptionalLong findFirst()
```
Returns an `OptionalLong` describing the first element of this
stream, or an empty `OptionalLong` if the stream is empty. If the
stream has no encounter order, then any element may be returned.This is a short-circuiting
terminal operation.
Returns:
an `OptionalLong` describing the first element of this
stream, or an empty `OptionalLong` if the stream is empty
