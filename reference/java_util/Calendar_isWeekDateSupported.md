#### isWeekDateSupported

```
public boolean isWeekDateSupported()
```
Returns whether this `Calendar` supports week dates.The default implementation of this method returns `false`.
Returns:
`true` if this `Calendar` supports week dates;
`false` otherwise.
Since:
1.7
