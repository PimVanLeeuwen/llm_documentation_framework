#### isWeekDateSupported

```
public final boolean isWeekDateSupported()
```
Returns `true` indicating this `GregorianCalendar`
supports week dates.
Overrides:
`isWeekDateSupported` in class `Calendar`
Returns:
`true` (always)
Since:
1.7
