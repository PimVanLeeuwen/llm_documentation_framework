```
@FunctionalInterface
public interface DoubleSupplier
```
Represents a supplier of `double`-valued results. This is the
`double`-producing primitive specialization of `Supplier`.There is no requirement that a distinct result be returned each
time the supplier is invoked.This is a functional interface
whose functional method is `getAsDouble()`.
Since:
1.8
See Also:
`Supplier`
